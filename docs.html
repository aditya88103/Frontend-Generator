⚡ Quick Start

FrontendGen is a browser-based tool. No installation, no server, no npm. Just open index.html in any modern browser and start generating.

💡 Tip

Use Chromium-based browsers (Chrome, Edge, Brave) for best results. Firefox works too but may have minor CSS differences.

🔑 Getting a Gemini API Key

FrontendGen uses the Gemini 2.0 Flash API. You need a free API key from Google AI Studio.

  1. Go to aistudio.google.com/app/apikey
  2. Sign in with your Google account
  3. Click Create API Key
  4. Copy the key (starts with AIza...)
  5. Paste it in Step 1 of the generator
⚠️ Security Note

Your API key is stored only in your browser session — it's never sent to any server other than Google's API. Do not share your key publicly.

The free tier includes generous quota (60 requests/minute). For heavy use, enable billing in your Google Cloud project.

📋 Step by Step

1

Paste your Gemini API key

Enter your key in the API key field. It will be auto-filled if you've used the app before this session.

2

Describe your website

Type a detailed prompt. Mention sections, content, colors, and the vibe you want. The more specific, the better the output.

3

Select a framework

Choose between HTML/CSS/JS, React, Vue, Bootstrap, or Tailwind. Each generates different file types.

4

Pick a visual style

Choose a template (Modern Dark, Glassmorphism, Minimal Light, Neon Cyberpunk, Corporate). This controls the design vibe.

5

Generate & download

Click Generate (or press Ctrl+Enter). View the live preview, inspect code in tabs, then download files individually or as a ZIP.

⚙️ Frameworks Explained

🌐 HTML + CSS + JS

Generates 3 separate files: index.html, style.css, script.js. Open index.html directly in browser. No build step needed.

⚛️ React JSX

Generates App.jsx, App.css, and index.html. Uses React 18 + Babel Standalone via CDN. Open index.html directly — no npm needed.

💚 Vue.js 3

Generates App.vue, main.js, index.html. Vue 3 via CDN, Composition API. Just open index.html.

🅱️ Bootstrap 5

Uses Bootstrap 5 grid, components, and utilities via CDN. Custom overrides in custom.css. Great for rapid professional UI.

🌊 Tailwind CSS

All styling done with Tailwind utility classes via CDN. No CSS file generated. Smallest footprint. Fastest to load.

⬇️ Downloading Your Project

After generating, you'll see individual download buttons for each file type, plus a Download ZIP button that bundles everything with a README:

  • Individual buttons: Download one file at a time
  • ZIP: Downloads all files + a README.md with project info
  • Code tabs: View and copy any file's code directly
💡 Tip

For React/Vue CDN projects, open index.html directly. For Tailwind, internet is needed to load the CDN. For plain HTML — works fully offline!

🖼️ Gallery (Project History)

Every project you generate is automatically saved to your browser's localStorage. Visit the Gallery page to:

  • Browse all previously generated projects with thumbnail previews
  • Open a full-screen modal preview of any project
  • Delete individual projects or clear all at once
⚠️ Note

Projects are stored locally in your browser. Clearing browser data or switching browsers will remove them. Maximum 20 projects are kept at once.

❓ FAQ

Why is the generated code not perfect?+

Gemini AI generates code based on your prompt and style guide, but like all generative AI it's not always perfect. Refine your prompt with more specific details. You can also edit the generated code manually.

Can I re-generate with a different style without changing my prompt?+

Yes! Just change the template style or framework, then click Generate again. The same prompt will be used with the new style guide. Each generation is saved separately in the Gallery.

Does it work offline?+

The generator itself requires internet to call the Gemini API. However, plain HTML+CSS+JS generated files work fully offline. React, Vue, Bootstrap, and Tailwind require internet to load their CDN scripts.

How do I use the React output?+

Open index.html directly in your browser. It loads React and Babel via CDN, then compiles and runs App.jsx. No npm or build step needed. For production use, you'd set up a proper Vite/CRA project and copy the JSX code.

Can I use this to build production sites?+

The generated code is a great starting point! For production, you should review and customize the code, add proper security, integrate a real backend, and optimize assets. Think of it as an AI-powered design scaffold.

Why does Gemini sometimes mix CSS into HTML?+

LLMs occasionally deviate from the exact output format. The app has a fallback parser that extracts inline <style> and <script> tags from HTML and splits them into separate files automatically.
